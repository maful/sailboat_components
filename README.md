# Sailboat Components

ViewComponents for the Sailboat UI.

## Usage (IN PROGRESS)
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "sailboat_components"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install sailboat_components
```
